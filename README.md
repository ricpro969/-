## Описание программы 
Random Password Generator — это графическое приложение для создания безопасных паролей. Программа позволяет гибко настраивать длину и состав символов, а также автоматически сохраняет историю созданных паролей в файл JSON для последующего просмотра.
## Требования для запуска 
Что нужно установить на компьютер: - Python 3.10 или выше - Библиотеки: `pip install matplotlib requests` ---
